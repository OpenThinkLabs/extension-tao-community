Take a Tour

This section takes you on a short tour of TAO, giving you an overview of how to prepare and organise your assessments using TAO. There are links to the relevant sections of the User Guide which will give you more detailed information on how to set up your own assessments once you start working with TAO.

Why use TAO?

TAO helps you set up and organise all of your student assessments easily and efficiently, be they internal end-of-year or end-of-course exams, mid-term assessments, or short weekly tests. TAO provides an authoring tool with which you can create new tests quickly and easily, as well as a simple architecture with which you can navigate your resources in order to re-use existing tests or parts of tests, or add new assessment material to assessments you or other teachers have used on past occasions, or with other groups.

Putting an Assessment together

An assessment in TAO consists of several different building blocks: Interactions, Items and Tests.

An Interaction is the most basic unit in an assessment, and takes the form of a question (e.g. multiple choice question), or other task type (e.g fill-the-gap exercise). An Item is a set of interactions to be used together, along with any supporting material, and a Test is a group of items, together with information on how they are ordered and presented to the test-taker.

Now, let's walk through the steps of create an assessment and manage your assessment resources.

1. Check what test items are already available

There may already be test items prepared by other users which are available to you, or by yourself for previous assessments.

In TAO's Assessment Builder Bar , select Items and examine the test items that are already available in the Library on the left.

If you do not have enough ready-to-go items, then you will need to create new ones. TAO offers a solution with intuitively designed testing Item templates.

2. Create Items

TAO has plenty of easy-to-use Item templates to work with. To create a new item, click the Items icon in the Assessment Builder Bar.

This opens the Items page, which consists of an interface with three parts. On the left is the Library, where you can view the inventory of already existing items. On the right is the Properties Panel, which you will use to select certain settings for your components (items, interactions, and tests), such as settings concerning your chosen scoring method for example. In the center is the Canvas, and it is in this last part where Item authoring takes place.

This layout is a common feature which runs through the TAO architecture, whichever component you are working on.

See the Creating a new item section for more details.

3. Add Interactions to your item

Your new Item will consist of interactions. Authoring an item means adding interactions to it. Interactions include the following types: simple, text-based, graphic, and miscellaneous (called Common Interactions, Inline Interactions, Graphic Interactions, and Custom Interactions in TAO). For further information on these types, see the Interactions section.

The procedures used to create interactions differ slightly depending on the type. The relevant procedure is described in each interaction section in detail.

4. Use your items in a Test

Once you have populated your item with interactions, you will need to build it into a Test before you can use it in an assessment. A test can include one or more items.

To do this, go back to TAO's Assessment Builder Bar, and select Tests. You can add items to a Test by selecting them from the Test Library. See the Creating a new test section for more details.

5. Give your Test a trial run

This can be done through a dummy test-taker account that you would have set up at the start of the academic year. Doing a trial run ensures that everything will run as expected when the students go through their assessment. Once the Test has been checked and everything is working as expected, you'll next want to set up a Delivery.

6. Register Your Test-Takers

You probably have a roster of students attending your courses. These lists of student names need to be converted into registered Test-Takers for use in TAO. In most cases, the instructor or course administrator will register the Test-Takers in the assessment system. To do this is relatively painless and only has to be done once, before the first assessment.

To do this, you will first need to select Test-Takers in the Assessment Builder Bar. See the Creating test-takers section for more details.

7. Assign test-takers to Groups

After entering or uploading the Test-taker profiles of all your students in the TAO assessment system, you will need to organise them into groups, depending on which students are taking which assessments. It may be that an entire class of students is taking the same assessments, or it may be that you need to create smaller groups of test-takers for certain types of assessment.

To do this, you will first need to select Groups in the Assessment Builder Bar. See the Creating a new group section for more details.

8. Publish and Deliver Your Test

Before your students can actually sit the assessment you've prepared, it needs to be assembled as a Delivery. Assembled Deliveries govern when a test will be taken by which selected individual or groups of test-takers, and how long it will last. They take only a few moments to put together.

To do this, you will first need to select Deliveries in the Assessment Builder Bar. See the Creating a new delivery section for more details.

9. View Your Results

After the assessment is over, you will want to see how your test-takers did.

To do this, select Results in the Assessment Builder Bar. See the Exporting results section for more details.
